inherits - determine if an object inherits a given file

int inherits( string file, object obj );

inherits() returns 0 if obj does not inherit file, 1 if it inherits the most recent copy of file, and 2 if it inherits an old copy of file.

See also: deep_inherit_list , inherit_list
