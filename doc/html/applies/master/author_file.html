author_file - determine the author for a given object

string author_file (string file);

This routine is only used if PACKAGE_MUDLIB_STATS is used.

This function must exist in the master object. It is called by the author statistic functions in the driver to determine what author a given object should be associated with. This is totally arbitrary and up to the mudlib designers wishes. It should be noted that the author that the object is assigned to will receive "credit" for all of the objects behavior (errors, heart_beats, etc).

See also: author_stats , domain_stats , domain_file
