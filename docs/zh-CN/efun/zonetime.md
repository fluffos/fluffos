### 名称

    zonetime() - 返回指定时区的时间字符串

### 语法

    string zonetime(string timezone, int time);

### 描述

    返回指定时区 `timezone` 和时间 `time` 的日期和时间的字符串，参数 `time` 是一个时间戳。

### 参考

    time(3), time_expression(3), localtime(3), uptime(3)
