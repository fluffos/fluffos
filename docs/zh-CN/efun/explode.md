### 名称

    explode() - 分割一个字符串

### 语法

    string *explode( string str, string del );

### 描述

    使用字符串 `del` 分割字符串 `str`，并返回分割后字符串组成的数组。

### 示例

    explode(str, " ") 会返回一个由字符串 `str` 中所有单词组成的数组。

    sscanf(3), extract(3), replace_string(3), strsrch(3)
