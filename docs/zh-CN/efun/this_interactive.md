### 名称

    this_interactive() - 返回当前玩家对象

### 语法

    object this_interactive();

### 描述

    返回呼叫此函数的玩家对象。此函数总是返回原始的玩家对象，而不像 this_player() 会受到 enable_commands() 或 command() 影响。

### 参考

    this_object(3), this_player()
