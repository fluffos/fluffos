### 名称

    real_time() - 返回一个本地服务器时间戳

### 语法

    int real_time( void );

### 描述

    返回一个本地服务器时间戳，时间戳是指格林威治时间1970年01月01日00时00分00秒(北京时间1970年01月01日08时00分00秒)起至现在的总毫秒数。

### 参考

    ctime(3), localtime(3), time(3), uptime(3)
