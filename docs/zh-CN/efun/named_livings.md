### 名称

    named_livings() - 返回所有已取名的生物对象

### 语法

    object *named_livings();

### 描述

    返回游戏中所有呼叫过 enable_commands() 和 set_living_name() 外部函数的对象的数组。

### 参考

    livings(3), set_living_name(3), enable_commands(3)
