### 名称

    member_array()  -  返回指定元素在数组或字符串中的索引位置

### 语法

    int member_array( mixed item, mixed * | string arr, void | int start );

### 描述

    返回 `item` 在 数组 `arr` 中首次出现的位置，如果指定起始位置 `start`，则从指定位置之后开始查询，如果没有找到返回 -1。
