### 名称

    all_previous_objects() - 返回呼叫当前函数的所有对象组成的数组

### 语法

    object *all_previous_objects();

### 描述

    返回一个由所有调用当前函数的对象组成的数组。注意：调用局部函数不会改变当前对象的 previous_object()。
