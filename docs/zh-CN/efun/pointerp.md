### 名称

    arrayp() / pointerp() - 判断一个给定变量是否为数组。

### 语法

    int arrayp(mixed arg);
    int pointerp(mixed arg);

### 描述

    如果变量 `arg` 是数组则返回1，否则返回0。

### 返回值

    1 或者 0
