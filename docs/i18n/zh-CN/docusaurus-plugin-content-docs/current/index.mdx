# FluffOS

FluffOS 是一个高性能游戏引擎（LPMUD 驱动），常用于构建 MUD —— 参见
[LPMud](https://en.wikipedia.org/wiki/LPMud)。FluffOS 最初是针对 MudOS 最后一个版本的补丁集，
经过多次大规模重写和缺陷修复，在支持新特性的同时保持了对现有 mudlib 的向后兼容。

代码库由三部分组成：

- **LPC 解释器** —— 解析并执行 LPC 程序。
- **网络服务器** —— 管理客户端连接并暴露给 LPC，支持 TELNET、TLS 和 WebSocket 协议。
- **粘合接口**
  - **Efun** —— LPC 程序可调用的内置 C/C++ 函数。
  - **Apply** —— 驱动在各类游戏事件发生时回调 LPC 对象的函数。

:::tip[当前版本]
当前版本为 `master` 分支 —— CMake 构建系统，支持 Ubuntu、macOS、Windows（MSYS2/MinGW64）
以及 WebAssembly。参见 [GitHub 上的所有版本](https://github.com/fluffos/fluffos/releases)。
:::

## 特性

- 使用 CMake 在 Ubuntu、macOS 和 Windows 上原生构建 —— 另有可在浏览器中运行完整 mudlib 的 WebAssembly 目标。
- 原生 UTF-8，透明的输入/输出编码转换支持。（支持 Emoji！）
- WebSocket 支持！
- TLS 支持！
- 默认 SHA512 crypt。
