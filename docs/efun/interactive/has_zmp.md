# has_zmp

### NAME

    has_zmp

### SYNOPSIS

    int has_zmp(object user)

### DESCRIPTION

    Return non-zero if interactive user has the ZMP protocol enabled in their
    client. 0 will be returned if the user does not.

    has_gmcp(), has_mxp()
