### NAME

    shout() - sends a message to all living objects

### SYNOPSIS

    void shout( string str );

### DESCRIPTION

    Sends the string 'str' to all living objects except this_player().

### SEE ALSO

    message(3), write(3), tell_object(3), tell_room(3), say(3)
