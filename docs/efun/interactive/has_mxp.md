# has_mxp

### NAME

    has_mxp

### SYNOPSIS

    int has_mxp(object user)

### DESCRIPTION

    Return non-zero if interactive user has the MXP protocol enabled in 
    their client. 0 will be returned if the user does not.

### SEE ALSO

    has_gmcp(), has_zmp()
