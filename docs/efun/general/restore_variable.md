### NAME

    restore_variable() - restore value of a variable from a string

### SYNOPSIS

    mixed restore_variable( string value );

### DESCRIPTION

    Restore  values  of  a  variable from a string.  The format used is the
    same format as save/restore_object.

### SEE ALSO

    restore_object(3)
