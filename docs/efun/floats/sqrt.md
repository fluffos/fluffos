### NAME

    sqrt() - returns the square root of a float

### SYNOPSIS

    float sqrt( float f );

### DESCRIPTION

    sqrt()  returns the non-negative square root of its argument, 'f'.  The
    value of 'f' must not be negative.

### SEE ALSO

    exp(3), log(3), pow(3)
