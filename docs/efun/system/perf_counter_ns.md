### NAME

    perf_counter_ns() - gives epoch time measurement in nanoseconds

### SYNOPSIS

    int perf_counter_ns( void );

### DESCRIPTION

    Return the number of nanoseconds since midnight (GMT) January 1, 1970.

    ctime(3), localtime(3), time_expression(3), uptime(3)
