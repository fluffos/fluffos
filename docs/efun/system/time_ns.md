### NAME

    time_ns() - return the number of nanoseconds since January 1, 1970

### SYNOPSIS

    int time_ns( void );

### DESCRIPTION

    Return the number of nanoseconds since midnight (GMT) January 1, 1970.

### SEE ALSO

    time(3), ctime(3), localtime(3), time_expression(3), uptime(3)
