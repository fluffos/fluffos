### NAME

    uptime()  -  return the number of seconds elapsed since the last driver
    reboot

### SYNOPSIS

    int uptime( void );

### DESCRIPTION

    This function returns the number  of  seconds  since  the  last  driver
    reboot.

### SEE ALSO

    time(3), ctime(3), localtime(3), time_expression(3)
