### NAME

    db_close() - close the database connection

### SYNOPSIS

    int db_close( int handle );

### DESCRIPTION

    Closes the database connection represented by the given handle.

    Returns 1 on success, 0 otherwise

    db_connect(3), db_status(3), valid_database(4)
