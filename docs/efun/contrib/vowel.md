# vowel

### NAME

    vowel() - test whether a character is a vowel

### SYNOPSIS

    int vowel( int c );

### DESCRIPTION

    Returns 1 if the character code `c` is an ASCII vowel
    (a, e, i, o, u in either case), 0 otherwise. Used together with
    add_a() for article selection.
