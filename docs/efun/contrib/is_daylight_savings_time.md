# is_daylight_savings_time

### NAME

    is_daylight_savings_time

### SYNOPSIS

    int is_daylight_savings_time(string, int);

### DESCRIPTION

    returns 1 if given time in given timezone is within daylight saving region
    0 if not
    -1 on error
