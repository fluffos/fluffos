### NAME

    upper_case

### SYNOPSIS

    string upper_case(string text);

### DESCRIPTION

    Returns a new string with every character in <text> converted to upper case.

### EXAMPLE

    string text = upper_case( "I like cheese." ) ; // "I LIKE CHEESE."

### SEE ALSO

    capitalize(3), lower_case(3)
