### NAME

    rtrim() - Remove trailing whitespaces (and others)

### SYNOPSIS

    string rtrim( string str );
    string rtrim( string str, string ch);

### DESCRIPTION

    Remove all trailing whitespace (or other characters 'ch') from the
    string 'str' and return a new string.

### SEE ALSO

    trim(3), ltrim(3)
