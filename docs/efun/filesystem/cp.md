### NAME

    cp() - copy a file

### SYNOPSIS

    int cp( string src, string dst );

### DESCRIPTION

    Copies the file <src> to the file <dst>.

### RETURN VALUE

    Returns 1 for success, returns a negative value in case of failure.

### SEE ALSO

    rm(3), rmdir(3), rename(3), link(3)
